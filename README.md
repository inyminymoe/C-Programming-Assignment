# C프로그래밍 중간과제물

### 예제 1: 헤더파일 분리
중복 코드를 헤더파일로 분리하여 모듈화

### 예제 2: 자료형 한계값
limits.h와 float.h를 사용한 자료형 범위 확인

### 예제 3: 전역/지역 변수
변수 스코프에 따른 동작 차이 분석

### 예제 4: Printf 형식지정자
다양한 출력 형식 연습

## 컴파일 방법
```bash
# 예제 1
gcc -o example1 main.c func_f.c func_g.c

# 예제 2
gcc -o int_limits int_limits.c
gcc -o float_limits float_limits.c
```
